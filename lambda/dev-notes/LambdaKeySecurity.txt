# 🔐 Lambda API Key Security

**Task:** Secure OpenWeatherMap API key using AWS Lambda environment variable.

---

## ✅ What Was Done

Replaced the hardcoded API key in the Lambda Python function with a secure environment variable to improve security and maintainability.

---

## 🔧 Steps Taken

1. Opened **AWS Lambda Console** and selected the function `getWeatherData`
2. Went to **Configuration → Environment Variables**
3. Added:
   - **Key:** `WEATHER_API_KEY`
   - **Value:** `******` (removed for security)
4. Updated the Python code:
```python
import os
api_key = os.environ.get("WEATHER_API_KEY")
```
5. Removed the hardcoded key from the script

---

## ✅ Benefits

- 🔐 API key is no longer exposed in code or GitHub
- 🔄 Easily updatable through AWS Console without modifying code
- ✅ Clean, secure, and production-ready practice

---

📌 This configuration ensures the weather service integration remains secure and scalable.
